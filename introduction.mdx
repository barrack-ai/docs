## First Steps

Recommended setup process:

<Steps>
  <Step title="Create Account">
    Create an account using email/password or Google OAuth
  </Step>
  <Step title="Complete Profile">
    Complete your profile in **My Account** section (required for payments)
  </Step>
  <Step title="Add Credits">
    Add credits to your account in the **Credits** section
  </Step>
  <Step title="Setup SSH Keys">
    Create or import an SSH key from the **SSH Keys** section
  </Step>
  <Step title="Deploy VM">
    Deploy your first virtual machine using the **Deploy VM** wizard
  </Step>
</Steps>
