# Account Management

Your account profile contains essential information for using the platform and processing payments.

## Personal Information

Update your basic personal details in your [account settings](https://barrack.ai/account):

<CardGroup cols={2}>
  <Card title="Basic Details" icon="user">
    **Required Information**
    - **Name:** Full name (required, letters, spaces, hyphens, apostrophes only, 2-100 characters)
    - **Email:** Your registered email address (cannot be changed after registration)
    - **Phone Number:** Optional contact number (international format supported)
  </Card>
  
  <Card title="Contact Preferences" icon="envelope">
    **Communication Settings**
    - **Primary Email:** Used for all platform communications
    - **Phone Notifications:** Optional SMS alerts for important events
    - **Marketing Preferences:** Control promotional communications
  </Card>
</CardGroup>

### Name Requirements

<Warning>
**Name Field Validation:**

- Required field with 2-100 characters
- Only letters, spaces, hyphens, and apostrophes are allowed
- Must match billing/legal name for payment processing
- Cannot be empty or contain numbers or special characters
</Warning>

### Email Address Management

<AccordionGroup>
  <Accordion title="Email Restrictions">

  <Accordion title="Email Verification">

## Billing Address

Complete your billing address to enable payments on the platform:

<Tabs>
  <Tab title="Required Fields">
    **Mandatory Information**
    - **Address Line:** Street address, apartment, suite, etc. (required, max 255 characters)
    - **Postal Code:** Postal or ZIP code (required, alphanumeric with hyphens/spaces)
    - **Country:** Select your country from the dropdown list (required)
  </Tab>
  
  <Tab title="Optional Fields">
    **Additional Information**
    - **City:** City name (optional, letters, spaces, hyphens, periods, apostrophes only)
    - **State/Province:** State or province name (optional, letters, spaces, hyphens only)
    - **Address Line 2:** Additional address information (apartment number, suite, etc.)
  </Tab>
  
  <Tab title="Validation Rules">
    **Address Validation**
    - **Address Line:** Required field with maximum 255 characters
    - **City:** Only letters, spaces, hyphens, periods, and apostrophes allowed
    - **State/Province:** Only letters, spaces, and hyphens allowed
    - **Postal Code:** Required, accepts alphanumeric characters, hyphens, and spaces
    - **Country:** Required selection from provided list
  </Tab>
</Tabs>

### Address Formatting Guidelines

<Steps>
  <Step title="Street Address">
    Enter your complete street address including:
    - House/building number
    - Street name
    - Apartment/suite number (if applicable)
    - Any additional location identifiers
  </Step>
  <Step title="Geographic Information">
    Provide accurate geographic details:
    - Correct city name
    - State or province (where applicable)
    - Accurate postal/ZIP code
    - Select correct country from dropdown
  </Step>
  <Step title="Validation">
    Ensure all information is accurate:
    - Double-check spelling of all fields
    - Verify postal code format for your country
    - Confirm country selection matches your location
    - Review for completeness before saving
  </Step>
</Steps>

## Form Validation

Input validation requirements ensure data quality and security:

### Field-Specific Validation

<Tabs>
  <Tab title="Name Validation">
    **Full Name Requirements**
    - **Character Set:** Only letters, spaces, hyphens, and apostrophes
    - **Length:** Between 2 and 100 characters
    - **Required:** Cannot be empty
    - **Format:** Proper name format expected
    - **Examples:** "John Smith", "Mary-Jane O'Connor", "José García"
  </Tab>
  
  <Tab title="Phone Validation">
    **Phone Number Format**
    - **Optional Field:** Not required for account completion
    - **Format:** Valid international format with country code
    - **Characters:** Digits, plus sign, spaces, dashes, parentheses only
    - **Length:** Reasonable length for international numbers
    - **Examples:** "+1 (555) 123-4567", "+44 20 1234 5678"
  </Tab>
  
  <Tab title="Address Validation">
    **Address Field Requirements**
    - **Address Line:** Required, maximum 255 characters
    - **City:** Optional, letters/spaces/hyphens/periods/apostrophes only
    - **State:** Optional, letters/spaces/hyphens only
    - **Postal Code:** Required, alphanumeric with hyphens/spaces
    - **Country:** Required selection from dropdown list
  </Tab>
</Tabs>

## Account Security

### Security Best Practices

<Steps>
  <Step title="Strong Password">
    - Use a unique, complex password
    - Include uppercase, lowercase, numbers, and symbols
    - Avoid personal information or common words
    - Change password regularly
  </Step>
  <Step title="Email Security">
    - Keep email account secure with strong password
    - Enable two-factor authentication on email if available
    - Monitor for suspicious login attempts
    - Keep email address current and accessible
  </Step>
  <Step title="Account Monitoring">
    - Review account activity regularly
    - Monitor credit usage and spending patterns
    - Check for unauthorized access or changes
    - Report suspicious activity immediately
  </Step>
  <Step title="Information Protection">
    - Keep billing information current and accurate
    - Protect account credentials from sharing
    - Use secure networks for account access
    - Log out from shared computers
  </Step>
</Steps>
